## 下载node包安装
切换到`~`目录
```
cd ~
```

可以在[https://nodejs.org/dist](https://nodejs.org/dist)地址中找到版本下载，例如：
```
wget https://nodejs.org/dist/v18.14.0/node-v18.14.0-linux-x64.tar.gz
```

此时当前文件夹下会有一个node-v18.14.0-linux-x64.tar.gz文件，进行解压
```
tar xvf node-v18.14.0-linux-x64.tar.gz
```

文件夹取个通用的名字
```
mv node-v18.14.0-linux-x64 node
```

然后就是配置环境变量了，首先执行`vi ~/.bashrc`(看你用的哪个)，在最下面加上：
```
export NODEJS_HOME=~/node
export PATH=$NODEJS_HOME/bin:$PATH
```
执行`source ~/.bashrc`应用配置

然后我们确认一下：
```
$ echo $PATH
/home/s517050/node/bin:/home/s517050/.local/bin:/home/s517050/bin:/usr/share/Modules/bin:/usr/local/bin:/usr/bin:/usr/local/sbin:/usr/sbin
```
可以看到前面出现了我们配置的路径。

此时可以试试`node`和`npm`命令，还没有的话可以重启下终端试试。
